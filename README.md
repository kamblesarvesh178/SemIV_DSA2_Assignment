## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/kamblesarvesh178/SemIV_DSA2_Assignment.git
   ```
2. Compile and run C++ files:
   ```bash
   g++ filename.cpp -o output
   ./output
   ```
3. Open `.ipynb` files using Jupyter Notebook or Google Colab.
